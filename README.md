# APEX API Docs Template

A Vercel-ready template for stunning, auto-generated API documentation. Integrate your Postman Collection or OpenAPI spec and deploy in minutes!

## Features

*   **Modern UI:** Clean, responsive design built with Tailwind CSS.
*   **Easy Deployment:** One-click deploy to Vercel.
*   **Swagger UI Integration:** Powers interactive API exploration.
*   **Customizable:** Adapt to your brand and needs.

## Getting Started

1.  **Fork/Clone** this repository.
2.  **Replace** `https://petstore.swagger.io/v2/swagger.json` in `index.html` with your OpenAPI spec URL or converted Postman collection URL.
3.  **Deploy to Vercel** using the 'Deploy' button or command line.
